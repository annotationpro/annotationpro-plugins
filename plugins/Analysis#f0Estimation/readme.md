# Analysis → f0Estimation

## Purpose

Estimates the fundamental frequency (F0) contour from speech recordings.

## Menu location

Analysis → f0Estimation

## Author

Katarzyna Klessa

## Description

This plugin is part of the Annotation Pro plugin collection.

## Acknowledgements

This plugin uses the F0 estimation module available in Annotation Pro.

The underlying implementation builds on a Java F0 extraction class developed by Hanna Kuczka using the pYIN approach described by Mauch and Dixon (2014), which served as the starting point for the development of the corresponding C# implementation in Annotation Pro.
